# CAIMEO-Sovereignty-Core

# Overview
This repository serves as the technical framework for the migration and reclamation of the Clarion Cognitive Networks intellectual property. It establishes the transition of the core cognitive architecture from the research environment at Rensselaer Polytechnic Institute (RPI) to the sovereign production framework of CAIMEO.NET.

# Objective
The goal of this project is to ensure the continuity, integrity, and independence of the Clarion logic. By decoupling the cognitive models, distributed tables, and network topologies from the legacy institutional infrastructure, CAIMEO.NET establishes itself as the primary host for the next generation of distributed intelligent systems.

# Migration Status
This repository contains the F# migration engine used to map, verify, and re-host legacy assets.

# Legacy: RPI Research Environment (Decommissioned) | Sovereign: CAIMEO.NET Production Framework (Active)

This repository is a public declaration of technical sovereignty and the official repository for the CAIMEO.NET ecosystem.
